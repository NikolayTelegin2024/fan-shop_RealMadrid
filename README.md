The Russian equivalent of the official Real Madrid store

2. Тематика сайта, название

Тематика: учебный веб‑сервис — прототип фан‑шопа ФК «Реал Мадрид», демонстрирующий навыки серверной разработки на Flask. Может использоваться фанатами клуба для ознакомления с ассортиментом атрибутики.

Название сайта: «Реал Мадрид Россия: Официальный фан‑шоп» (учебный проект).

3. Цель и задачи системы

Цель: разработка веб‑сервиса с использованием Flask и сопутствующих технологий для получения практических навыков серверной разработки в рамках единого проекта. Проект также может служить демонстрационным ресурсом для фанатов клуба.

Задачи:

    Реализовать серверную часть на Flask с шаблонами Jinja2.
    
    Настроить работу со статическими файлами (CSS, JS, изображения атрибутики клуба).

    Разработать формы для взаимодействия пользователей (регистрация, авторизация, корзина).

    Создать базу данных для хранения информации о товарах, пользователях, заказах.

    Реализовать систему аутентификации пользователей.

    Разработать REST API для взаимодействия фронтенда и бэкенда.

    Обеспечить адаптивный интерфейс с демонстрационным функционалом (без реальной оплаты).

    Создать контент, интересный для фанатов (новости клуба, фотогалереи, статистика).

5. Содержание сайта и структура БД
    Структура базы данных (SQLite):

        Users (id, username, email, password_hash, is_admin, created_at)

        Products (id, name, price, category, image_url)

        Cart (id, user_id, product_id, quantity)

        Order (id, user_id, order_date, delivery_date, total_price)

        OrderItem (id, order_id, product_name, product_price, quantity)

Основные разделы сайта:

    Главная страница (/):

        список всех товаров с фото, названием, ценой и категорией

        ссылка на карточку товара

    Карточка товара (/product/<id>):

        фото товара, название, цена, категория

        выбор количества (от 1 до 99)

        кнопка «Добавить в корзину»

    Корзина (/cart):

        список выбранных товаров с количеством и ценой

        удаление товаров из корзины

        расчёт итоговой стоимости

        кнопка «Оформить заказ»

    Авторизация (/login, /register):

        формы с валидацией (Flask-WTF)

        хеширование паролей (Werkzeug)

        сессия пользователя (Flask-Login)

    Личный кабинет (/profile):

        профиль пользователя (имя, email, дата регистрации)

        история заказов с датой доставки (+10 дней)

        список товаров в каждом заказе

    Админ-панель (/admin):

        просмотр всех товаров в табличном виде

        добавление нового товара (название, цена, категория, ссылка на фото)

        редактирование существующих товаров

        удаление товаров

    REST API:

        GET /api/products — список всех товаров (JSON)

        GET /api/products/<id> — конкретный товар

        POST /api/products — создание товара

        PUT /api/products/<id> — обновление товара

        DELETE /api/products/<id> — удаление товара

6. Функциональные возможности и технологии
    1) Серверная часть (Flask):

        Маршрутизация: Flask routes для всех разделов

        Шаблоны: Jinja2 для динамического контента (наследование шаблонов, блоки)

        Статические файлы: CSS (стилизация) в папке static

        Формы: Flask-WTF для регистрации, авторизации, добавления и редактирования товаров с CSRF-защитой

        Аутентификация: Flask-Login для сессий, хеширование паролей (Werkzeug)

        База данных: SQLite + SQLAlchemy ORM

        REST API: эндпоинты /api/products и /api/products/<id> с методами GET, POST, PUT, DELETE

        Разграничение ролей: обычные пользователи и администраторы (is_admin)

        Обработка ошибок: кастомные страницы 404, 403, 500

    2) Клиентская часть:

        Адаптивный дизайн (CSS Flexbox, медиа-запросы)

        Формы с серверной валидацией и выводом flash-сообщений

        Выбор количества товара при добавлении в корзину

        Отображение даты регистрации и даты доставки в формате ДД.ММ.ГГГГ

    3) Административная часть:

        Вход в админ-панель только для пользователей с правами admin

        Полный CRUD товаров (создание, чтение, обновление, удаление)

        Табличное отображение всех товаров с фото, ценой и категорией

        Подтверждение удаления товара

    4) Пользовательская часть:

        Регистрация с указанием имени, email и пароля

        Вход с проверкой логина и пароля

        Добавление товаров в корзину с выбором количества

        Просмотр корзины с итоговой суммой

        Оформление заказа с автоматическим расчётом даты доставки (+10 дней)

        История заказов в личном кабинете





Чтобы установить и запустить проект, клонируйте репозиторий, 
создайте виртуальное окружение командой python -m venv venv, 
активируйте его (venv\Scripts\activate для Windows или source venv/bin/activate для Mac/Linux), 
установите зависимости через pip install -r requirements.txt, 
затем выполните python app.py и откройте в браузере http://127.0.0.1:5000.
